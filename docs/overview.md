# 📸 Screenshot Share App - サービス概要

EpicSnap は、ゲーマー向けに設計された**スクリーンショット共有アプリ**です。

## 🎮 主な特徴

- **保存数制限**：無料プランでは 20 枚まで、有料プランで無制限に保存可能
- **共有アルバム機能**：友達とスクリーンショットを共有するためのアルバムを作成・参加可能
- **シンプルな UI**：直感的な操作で簡単にスクリーンショットを管理
- **高速なアップロード**：Supabase Storage を活用した高速なアップロード体験
- **マルチデバイス対応**：Web ブラウザからいつでもアクセス可能

## 👥 対象ユーザー

- ゲームのスクリーンショットを共有したいゲーマー
- チームでゲームプレイを記録・共有したい e スポーツプレイヤー
- ゲームの思い出を残したいカジュアルゲーマー

## 💰 料金プラン

| プラン | 保存可能枚数 | 共有アルバム | 広告 |
| ------ | ------------ | ------------ | ---- |
| 無料   | 20 枚        | 制限あり     | あり |
| 有料   | 無制限       | 全機能利用   | なし |

## 🛠️ 技術的特徴

- **フロントエンド**: React / Next.js
- **バックエンド**: Supabase (Auth, Database, Storage)
- **認証**: Supabase Auth
- **課金システム**: Stripe 連携
- **ホスティング**: Vercel
